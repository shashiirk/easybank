See it live [here](https://shashiirk.github.io/easybank).

## Get Started

Clone this repository to your local machine

```
$ git clone https://github.com/shashiirk/easybank.git
```

Install all the dependencies

```
$ npm install
```

Start the application

```
$ npm run dev
```
